## 🛠️ 사용 기술 및 라이브러리

- PHP/Codeigniter
- HTML/CSS/JS
- MySQL
- BootStrap
- Naver Cloud Server

- 회원 가입/로그인 기능
- 게시판 글 작성, 수정, 삭제, 조회 기능 (자유, 문의 게시판)
- 댓글, 검색 기능
- 수강 과목 평가 조회 및 등록 기능
